# log
## 特性
`pkg/log` 是基于 `zap` 包封装一个日志包。具有如下特性：

- 支持日志级别：`Debug`、`Info`、`Warn`、`Error`、`Panic`、`Fatal`。
- 支持自定义配置。
- 支持文件名和行号。
- 支持输出到标准输出（stdout）和文件，可以同时输出到多个地方。
- 支持 `JSON` 和 `Text` 两种日志格式。
- 支持颜色输出。
- 兼容标准的 `log` 包。
- 支持结构化日志记录。
- 支持日志滚动
- 支持日志文件压缩
- 支持动态调整日志级别
